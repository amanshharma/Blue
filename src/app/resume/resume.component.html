Web based control panel for MS LYNC

This is an internal tool that is being used by Infosys MS Odc Team. The current version provides the below mentioned features: • Enable/Disable users from Lync setup • View Information of an enabled user • Change configuration of a user by comparing with old configuration side by side



An Interactive web Interface to Triage customer cases

This web application helps the customers to narrow down the case problem by providing an interactive solution based interface. This app helps in reducing the number of cases raised by customers. Current implementation handles IMnP (Instant Messaging and Presence) cases



Centralized Deployment Feature for Cisco IMnP (Instant Messaging and Presence)

This feature involved the redesigning of the basic architecture of IMnP for a specific client. In basic design we have n number of IMnP clusters for n number of CUCM clusters. With This feature n number of IMnP clusters are reduced to 1 Centralized cluster that will interact with all CUCM Clusters and enables the efficient Instant messaging and rich presence
